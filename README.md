# Ruby Lisp Parser 

A lisp parser made in Ruby that transforms generic Lisp code into an AST.

## Features

- Easily extensible with reader macros.
- Clojure-like #() function sugar.
- Parses escaped strings.

## Instructions 

### Installation
```
bundle install
```

### Launching

```
ruby main.rb
```

### Launching tests 

```
spec
```
